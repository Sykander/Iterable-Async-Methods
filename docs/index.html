Iterable Async

A collection of methods for looping iterable objects asynchronously using something similar to the Array api.

Installation

 $ npm install iterable-async

Dependencies

  • None

Supports

  • JavaScript
  • TypeScript

Usage

ES6 Format

const {
	asyncFind,
	asyncFindIndex,
	asyncFilter,
	asyncForEach,
	asyncMap,
	asyncMapSort,
	asyncReduce,
	asyncSort
} = require('iterable-async');

TypeScript Format

import {
	asyncFind,
	asyncFindIndex,
	asyncFilter,
	asyncForEach,
	asyncMap,
	asyncMapSort,
	asyncReduce,
	asyncSort
} from "iterable-async";
